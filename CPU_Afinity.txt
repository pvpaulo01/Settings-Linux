Configurando CPU Afinity


Olá iremos configurar o CPU Afinity para uma melhor distribuções de processos entre os processadores atuais das maquina.

1° Passo iremos verificar o PID do processo do serviço HTTP
	#pidof httpd


2º Passo iremos definir a numeração do processo para a CPU desejada.
 	 #taskset <Options> <Numbercpu> <NumberPID>
	 #taskset -cp 0,1 4212
